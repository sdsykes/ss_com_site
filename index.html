TotalSpaces

I stopped development of TotalSpaces ("The ultimate grid spaces manager for your Mac") after it became too difficult to inject code into Dock.app, and the work-arounds that I found did not work well enough for a production quality product. The web page remains here and the latest TS3 beta is here which used to work to some extent on Apple Silicon macs. No longer supported in any way.
